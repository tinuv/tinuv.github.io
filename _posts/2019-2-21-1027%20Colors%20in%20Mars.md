# 题目
[题目链接](https://pintia.cn/problem-sets/994805342720868352/problems)

这个题目的意思是类似于人类的颜色表示法,火星人也用数字来表达颜色,只不过,它们不用16进制,而是使用13进制,本质上就是一个进制转换的问题

# 知识点

## 底数大于10的进制转化
由10进制转化为k(k>10)进制的数总体上来讲是使用"除基取余法",但需要做一些改动,因为大于10的数通常还会加入一些字母,这些字母需要额外处理,比如判断余数来决定是否需要使用字母.


# 代码
```cpp
#include <iostream>
#include <map>
#include <string>
#include <algorithm>

using namespace std;

int main() {
    int color[3];
    cin>>color[0]>>color[1]>>color[2];
    string result = "#";
    map<int,string> dict = {{10,"A"}},{11,"B"},{12,"C"}};
    for(int i=0; i<3; i++) {
        string a = "";
        while(color[i]!=0) {
            int temp = color[i]%13;
            a = a+(temp>9?dict[temp]:to_string(temp));
            color[i] = color[i]/13;
        }
        if(a.size()==0)
            a = "00";
        else if(a.size()==1){
            a.insert(0,"0");
            reverse(a.begin(),a.end());
        }
        result = result+a[1];
        result = result+a[0];
    }
    cout<<result;
    return 0;
}

```
