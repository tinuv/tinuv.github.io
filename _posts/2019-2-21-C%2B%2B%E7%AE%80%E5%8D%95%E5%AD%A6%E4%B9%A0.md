# 数组
C++支持数组,用法与Java语言的用法基本一样
## 一维数组
```c++
#include <iostream>

using namespace std;

int main() {
    int a[5] = {1, 2, 3, 4, 5};
    cout << a[0];
    return 0;
}
```
## 多维数组
```cpp
#include <iostream>

using namespace std;

int main() {
    int a[2][2] = {{1, 2}}, {1, 2}};
    cout << a[0][0];
    return 0;
}
```
## 小技巧:将数组初始化为0
```c++
#include <iostream>

using namespace std;

int main() {
    int a[2][2] = {0};
    for(int i = 0; i < 2; i++) {
        for(int j = 0; j < 2; j++) {
            cout << a[i][j];
        }
    }
    return 0;
}
```

## 小技巧:计算数组的长度
不像java的数组是面向对象的,c++也没有类似于python的len()方法,c++计算数组的长度比较原始,即使用元素的size除以总的size,使用sizeof()函数可以返回元素的大小和数组的大小.
```c++
#include <iostream>

using namespace std;

int main() {
    int a[2][2] = {0};
    int b;
    b = sizeof(a) / sizeof(a[0]);
    cout << b;
    return 0;
}
```

# 列表

## Vector
翻译过来是向量,实际上与数组相似,即连续储存而不是链式储存,vector是一个模板类,具有面向对象的特性,通过vector.size()可以获取其大小,他也有与数组一样的操作符[]来访问vector里面的元素,同样不同于Java语言的是,C++不用new来创建一个Vector对象,直接声明即可,在Java中如果只是声明是不会创建一个对象的.
```c++
#include <iostream>
#include <vector>

using namespace std;

int main() {
    vector<int> a;
    a.push_back(2);
    a.push_back(4);
    cout << a[1];
    return 0;
}
```
vector通过push_back()往最后追加一个元素.

## 访问第一个元素和最后一个元素
通过vector.front()访问第一个元素,通过vector.back()访问最后一个元素
```c++
#include <iostream>
#include <vector>

using namespace std;

int main() {
    vector<int> a;
    a.push_back(2);
    a.push_back(4);
    cout << a.front() << "\n";
    cout << a.back() << "\n";
    return 0;
}
```

## 迭代器
迭代器的作用在于插入元素和删除元素的时候作为一个参数,有两个最常用的迭代器:vector.begin()和vector.end()
```c++
#include <iostream>
#include <vector>

using namespace std;

int main() {
    vector<int> a;
    a.push_back(2);
    a.push_back(4);
    a.insert(a.begin() + 1, 1, 10);
    a.insert(a.end() - 1, 1, 20);
    for(int i = 0; i < a.size(); i++) {
        cout << a[i] << " ";
    }
    return 0;
}
```
这两个往第二个和倒数第二个插入了两个数

## 删除元素
### 删除最后一个元素使用vector.pop_back()
```c++
#include <iostream>
#include <vector>

using namespace std;

int main() {
    vector<int> a;
    a.push_back(2);
    a.push_back(4);
    a.insert(a.begin() + 1, 1, 10);
    a.insert(a.end() - 1, 1, 20);
    a.pop_back();
    for(int i = 0; i < a.size(); i++) {
        cout << a[i] << " ";
    }
    return 0;
}
```
### 删除所有元素使用vector.clear()
```c++
#include <iostream>
#include <vector>

using namespace std;

int main() {
    vector<int> a;
    a.push_back(2);
    a.push_back(4);
    a.insert(a.begin() + 1, 1, 10);
    a.insert(a.end() - 1, 1, 20);
    a.clear();
    for(int i = 0; i < a.size(); i++) {
        cout << a[i] << " ";
    }
    return 0;
}
```
### 删除任意位置的一个元素使用vector.erase()
与插入一样需要一个迭代器
```c++
#include <iostream>
#include <vector>

using namespace std;

int main() {
    vector<int> a;
    a.push_back(2);
    a.push_back(4);
    a.insert(a.begin() + 1, 1, 10);
    a.insert(a.end() - 1, 1, 20);
    a.erase(a.begin() + 1);
    for(int i = 0; i < a.size(); i++) {
        cout << a[i] << " ";
    }
    return 0;
}
```
